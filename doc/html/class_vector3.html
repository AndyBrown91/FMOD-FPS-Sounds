Detailed Description

Simple 3D vector.

This has the same structure as an FMOD vector and an OpenGL Vector3 to make things easier to typedef and inetegrate as needed.

Just do: 

 typedef FMOD_VECTOR Vector3;
 #define VECTOR3

..before including GameEngineServer.h and Gamecon's own Vector3 won't be defined.
